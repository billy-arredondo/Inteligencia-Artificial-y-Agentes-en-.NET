# Políticas Avanzadas de IT  

---

## 1. Acceso a sistemas

- El acceso se otorga según el principio de **mínimo privilegio**.
- Los accesos deben ser aprobados por el líder del área.

---

## 2. Solicitudes de software

- Se debe crear un ticket para solicitar software adicional.
- IT validará licencias y compatibilidad.

---

## 3. Respaldo de información

- Los colaboradores deben guardar su trabajo en sistemas corporativos.
- No se permite almacenar información crítica en dispositivos personales.

---

## 4. Uso de dispositivos personales

- Está permitido bajo modalidad BYOD con restricciones.
- Deben cumplir políticas de seguridad (antivirus, cifrado, etc.).

---

## 5. Incidentes de seguridad

- Todo incidente debe reportarse inmediatamente.
- IT iniciará un proceso de análisis y mitigación.
